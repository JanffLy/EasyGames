# 🐍 贪吃蛇游戏 (Snake Game)

一个基于前后端分离架构的经典贪吃蛇游戏实现。

## 🎮 游戏介绍

这是一个现代化的贪吃蛇游戏实现，具有经典玩法的同时添加了排行榜、障碍物等增强功能。游戏采用Go语言开发后端，Vue 3开发前端，提供流畅的游戏体验。

## 💻 技术栈

- **后端**：
  - Go 语言
  - Beego 框架
  - MySQL 数据库
  - Redis 缓存
- **前端**：
  - Vue 3
  - Vite

## ✨ 游戏特性

- 经典贪吃蛇玩法
- 实时排行榜功能
- 自动生成障碍物增加挑战性
- 积分系统（时间*1 + 豆子数量*10）
- 响应式设计，支持不同设备

## 🚀 安装与启动

### 📋 环境要求

- Go 1.16+
- Node.js 16+
- MySQL
- Redis

### 🔧 后端启动步骤

1. 进入后端目录
```bash
cd ./snake/backend
```

2. 安装依赖
```bash
go mod tidy
```

3. 配置数据库连接
编辑 `conf/app.conf` 文件，设置MySQL和Redis连接信息：
```
mysql.user = root
mysql.password = your_password
mysql.db = snake_game

redis.host = 127.0.0.1
redis.port = 6379
```

4. 启动后端服务
```bash
go run main.go
```
后端服务默认运行在 http://localhost:8080

### 🌐 前端启动步骤

1. 进入前端目录
```bash
cd ./snake/frontend
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```
前端服务默认运行在 http://localhost:3001

4. 构建生产版本（可选）
```bash
npm run build
```

## 📜 游戏规则

1. **基本玩法**：控制蛇移动，吃到食物使蛇变长并得分
2. **控制方式**：使用WASD键控制蛇的移动方向
   - W - 向上
   - S - 向下
   - A - 向左
   - D - 向右
3. **游戏结束条件**：
   - 蛇头碰到墙壁
   - 蛇头碰到自己的身体
   - 蛇头碰到障碍物
   - 10秒内未吃到食物
4. **得分规则**：
   - 每存活1秒：+1分
   - 每吃到1个食物：+10分
5. **障碍物**：游戏过程中会随机生成障碍物，增加游戏难度

## 🎨 游戏界面

游戏界面包含以下元素：
- 游戏标题
- 游戏状态显示
- 游戏主界面（15x15的网格）
- 游戏信息面板（得分、时间、豆子数量）
- 控制按钮（开始游戏、保存得分、查看排行榜）
- 操作说明
- 排行榜对话框

## 🏆 排行榜功能

游戏支持保存得分记录并查看排行榜。游戏结束后，可以选择保存当前得分，然后在排行榜中查看所有玩家的最佳成绩。

排行榜显示以下信息：
- 排名
- 得分
- 游戏时长
- 豆子数量
- 游戏日期

## ⚙️ 配置说明

### 🖥️ 后端配置

主要配置位于 `conf/app.conf` 文件中：

- `httpport`：HTTP服务端口
- `wall.max`：最大障碍物数量（默认6个）
- `game.speed`：游戏速度（默认200毫秒）
- MySQL和Redis连接配置

### 🎯 游戏参数

- 游戏地图尺寸：15x15
- 蛇的初始长度：3
- 障碍物最大数量：6（可配置）
- 游戏更新间隔：200ms（可配置）

## ❓ 常见问题

### 🚨 后端服务启动失败

1. 检查数据库连接配置是否正确
2. 确保MySQL和Redis服务正在运行
3. 查看控制台错误信息

### 🔌 前端无法连接后端

1. 检查后端服务是否正常启动
2. 确认前端API调用地址是否正确
3. 检查是否存在跨域问题
